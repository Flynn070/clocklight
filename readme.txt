Small room monitor & clock using a raspberry pi pico H with an lcd readout and strip of 15 leds, the first 12 representing numbers on an analogue clock with 3 colours representing each clock hand.

uses micropython libraries: time, datetime, dht, ssd1306

- Raspberry pi pico H
- 16x2 lcd screen with i2c backpack
- 1-wire temperature sensor
- dht11 temperature & humidity sensor
- ds1302 RTC with pcb module
- 10k panel mount potentiometer
- Photo transistor (light sensor)
